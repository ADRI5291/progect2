Логіка це школа програмуваня, у ній можна навчитися багато чому.Я зараз вчуся на Вебдизайні.

Я тут вчу як створювати сайт. Також мову HTML,CSS.

теги html

  1. Щоб почати створеня сайту-- < html >
  2. Назва сторінки для вікна браузера-- < title >
  3. Між цими тегами увесь вміст сайту-- < body >
  4. Ось так пишуться теги "<...>" щоб закрити записуємо те саме але після 1 трикутної дужки пишемо"/"
  5. Між цими тегами писати те що треба користувачевів першу чергу логотип, заголовок-- < header >
  6. Для основної інформація-- < main >
  7. Підвал сайту-- < footer >
  8. Заголовок-- < h1 >
  9. Для абзацу-- < p >
  10. Картинка-- < img scr >
  11. Посилання-- < a href >

CSS

  • В мові CSS ми просто змінюємо те що створили за допомогою html. записуємотак- <...: ...> А при перечислені < ...: ... ; ...: ...; >
  • Щоб змінити колір фону треба-- < background-color >
  • Щоб змінити колір тексту треба-- < color >
  • Щоб змінити розташування текста/...-- < text-align >
  • Щоб змінити розмір тексту-- < font-size >
  • Об зробити зовнішні відступи-- < margin >
  • Щоб зробити внутрішні відступи треба-- < padding >
  • Щоб підкреслити текст-- < text-decoration >
  • Для картинок
  • Висота картинки--< height="...px" >
  • Висота картинки--< width="...px" >
  • Щоб заокруглити краї-- < borde-radius >
  • Загалом це те що я вивчив. Деякі я не записав
